## 🐛 Common Issues

### Issue: "Cannot connect to MongoDB"
**Solution**: Check `.env` file has correct `MONGO_URI`

### Issue: "Duplicate key error"
**Solution**: Run seed again (it clears data first), or manually drop collections

### Issue: "Password won't work for login"
**Solution**: Ensure you're copying the exact password from this file (watch for whitespace)

### Issue: "Drafts not appearing"
**Solution**: Try refreshing the browser or clearing cache, check user is logged in as the draft author
